# PROJECT CHARTER

**Project Name**: Management Event Kolam Renang (Digitalisasi Lomba Renang)
**Client / Owner**: Khafid Swimming Club
**Date**: 05 Februari 2026
**Version**: 1.0
**Status**: Draft

---

## 1. Latar Belakang & Masalah (Business Case)

Saat ini, proses manajemen dan pendaftaran event lomba renang di **Khafid Swimming Club** masih dilakukan secara manual/semi-manual.

**Masalah Utama (Pain Points):**

- Pendaftaran masih menggunakan Google Form, Manual, atau Chat WhatsApp.
- Data peserta berantakan dan tidak terpusat.
- Proses rekap data sulit dilakukan oleh panitia.
- Peserta sering bingung mengenai informasi dan status pendaftaran.

## 2. Tujuan Proyek (Project Goals)

Membangun sistem berbasis website untuk **Digitalisasi Total** proses pendaftaran dan manajemen event.

**Hasil Akhir yang Diharapkan:**

- **Data Rapi**: Sentralisasi data peserta dan event dalam satu database.
- **Proses Cepat**: Otomatisasi pendaftaran dan verifikasi.
- **Profesional**: Tampilan website yang modern (Sporty & Formal) meningkatkan citra klub.

## 3. Ruang Lingkup (Scope)

### In-Scope (Termasuk):

- **Halaman Publik**: Home, About (Visi Misi, Pelatih), Jadwal Event, Galeri, Kontak.
- **Sistem Pendaftaran**: Form registrasi atlet (Nama, Umur, Klub, KTP), Upload Bukti Bayar.
- **Dashboard Peserta**: History event, status pendaftaran (Pending/Diterima/Ditolak), Notifikasi.
- **Dashboard Admin**: Kelola event, validasi pembayaran, download rekap data (Excel).
- **Multi-Role**: Admin, Pelatih, Guest, Peserta.

### Out-of-Scope (Tidak Termasuk - Fase Selanjutnya):

- Mobile Apps (Android/iOS) - Fokus di Web Responsive dulu.
- Fitur Manajemen Atlit / Pelatihan Harian (Fokus ke Event Lomba dulu).

## 4. Timeline Target

- **Deadline Utama**: Sebelum Puasa 2026.
- **Milestone**: Event terdekat setelah Hari Raya harus sudah menggunakan sistem baru.
